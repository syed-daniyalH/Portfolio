## Projects Section

### Project 1: Dispatch Alex
**Subtitle**: AI-Powered Automotive CRM Automation
**Client**: Automotive Dealership

**Challenge**: Manual job assignment and customer communication was taking 10-15 minutes per job, creating bottlenecks in the dealership's workflow.

**Solution**: Built an intelligent SMS-to-action system using Make.com and FastAPI. SMS arrives via Twilio → backend pulls dealership data → OpenAI determines intent → automated reply sent → job auto-assigns technician → QuickBooks auto-invoices on completion.

**Impact**: 
- ⚡ 10–15 min → seconds per job assignment
- ⚡ Auto-invoicing on completion
- ⚡ Zero manual steps in workflow

**Tech Stack**: Make.com, FastAPI, PostgreSQL, React, TypeScript, QuickBooks API, Vercel, Render

---

### Project 2: SM2 Racing
**Subtitle**: Race Data Automation & OCR Pipeline
**Client**: Racing Team / Sports Analytics

**Challenge**: Race teams were spending 2+ hours manually transcribing race notes, extracting data, and updating reports.

**Solution**: Built an OCR + AI pipeline. Raw race notes + images → OCR extraction → OpenAI analysis → tire pressures, lap times, RR bound data extracted → written to Google Sheets → auto-posted to web app.

**Impact**:
- ⚡ 2 hours → under 5 minutes per event
- ⚡ 100% data accuracy with AI validation
- ⚡ Real-time report generation

**Tech Stack**: Make.com, OCR, OpenAI API, Google Sheets, Next.js, FastAPI, Vercel, Render

---

### Project 3: LinkedIn Content Automation
**Subtitle**: n8n AI Agent Workflow
**Client**: Content Creator / Personal Brand

**Challenge**: Creating consistent LinkedIn content was taking 3-4 hours per week of manual work.

**Solution**: Scheduled n8n workflow: research agent pulls trending topics via SerpAPI → content agent writes post + hashtags → DALL-E generates image → draft sent for approval → auto-posted to LinkedIn or regenerated if rejected.

**Impact**:
- ⚡ Saves 3–4 hours of content work per week
- ⚡ Consistent posting schedule maintained
- ⚡ AI-generated visuals matching brand

**Tech Stack**: n8n, OpenAI, DALL-E, SerpAPI, LinkedIn API
