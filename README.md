# Syed Muhammad Daniyal Haider Portfolio

A clean Vite + React + TypeScript portfolio for an AI Automation Engineer and Python Developer.

## Stack

- Vite
- React 19
- TypeScript
- Tailwind CSS v4
- Framer Motion
- Wouter
- Lenis
- React Hook Form
- Zod

## What It Includes

- Sticky navigation with section links
- Premium hero section with resume CTA
- Metrics, overview, about, skills, projects, experience, education, certifications, and contact sections
- Accessible contact form with validation
- SEO metadata, Open Graph tags, sitemap, and robots file
- Responsive layout tuned for desktop, tablet, and mobile

## Run Locally

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
```

## Notes

- This repo contains a static frontend portfolio.
- No custom backend is included in the project.
