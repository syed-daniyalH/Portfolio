# Syed Muhammad Daniyal Haider Portfolio

## Overview

This repository contains a professional developer portfolio website for an AI Automation Engineer and Full Stack Developer. The site is designed to present services, project work, technical strengths, and client value in a way that is clear for recruiters, business owners, and collaboration partners.

## Key Features

- Premium hero section with clear positioning and call to action
- About, skills, experience, education, and certification sections
- Featured project and case-study style presentation
- Contact form with validation
- Responsive layout for desktop, tablet, and mobile
- SEO essentials including Open Graph tags, sitemap, and robots file
- Smooth motion and polished front-end presentation

## Tech Stack

- Vite
- React 19
- TypeScript
- Tailwind CSS 4
- Framer Motion
- Wouter
- React Hook Form
- Zod
- Playwright

## Business Use Case

This portfolio works as a personal branding website for showcasing AI automation services, full stack development capability, workflow automation expertise, and business software projects to recruiters and potential clients.

## Setup

```bash
pnpm install
pnpm dev
```

Build for production:

```bash
pnpm build
```

## Project Status

Active portfolio website. The project is a polished static frontend focused on presentation, SEO, and lead generation rather than a custom backend product.
